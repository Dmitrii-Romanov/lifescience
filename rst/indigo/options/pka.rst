###########
pKa options
###########

.. indigo_option::
    :name: pKa-model
    :type: string
    :short: pKa model type
    :default: simple
    
    Sets type of pKa model. Available models:
    * simple
    * advanced

.. indigo_option::
    :name: pKa-model-level
    :type: int
    :short: max level for advanced model
    :default: 0

    Sets maximal level of advanced pKa model is used for pKa estimation.

.. indigo_option::
    :name: pKa-model-min-level
    :type: int
    :short: min level for advanced model
    :default: 0

    Sets minimal level of advanced pKa model is used for pKa estimation.
